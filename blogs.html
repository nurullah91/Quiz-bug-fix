1. What is the difference among map, filter, forEach and find?

We can throw map, filter, forEach, find to an array. map() gives us the same code on every element in that array and returns a new array with the updated elements. filter()checks every element in that array to see if its condition get true then it returns a new array with the elements that return truthy for the condition. forEach() is used to execute the same code on every element in an array but does not change the array and it does not return anything. find() checks the condition to that array and when it got value whatever is true for the condition then it will stop searching

3. What is Event loop and how it works?

Event Loop check the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and event loop will push it to the Call Stack. An iteration is called a tick in the Event Loop
